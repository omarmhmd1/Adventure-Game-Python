# Adventure Game in Python

مرحباً بك في لعبة **مغامرة الفضاء**! هذه لعبة تفاعلية تعتمد على النصوص مكتوبة بلغة Python. في هذه اللعبة، سيتعين عليك اتخاذ قرارات لتحديد مصيرك ومواجهة الفضائيين لإنقاذ الأرض.

## كيفية اللعب

- تبدأ اللعبة في مركز حماية الأرض ضد الكائنات الفضائية حيث تتلقى رسالة حول تهديد الفضائيين.
- يتعين عليك اتخاذ قرارات استراتيجية مثل:
  - هل ستواجه الفضائيين أم ستترك المهمة لشخص آخر؟
  - هل ستهاجمهم أو ستعود إلى مكان آمن؟
  - هل ستحل اللغز الذي سيعينك على تدمير المدفع الليزري الضخم؟

اللعبة تحتوي على سيناريوهات متعددة مع خيارات تؤثر على سير القصة ونهاية اللعبة. ستكون لديك محاولات متعددة لتحقيق النجاح أو الفشل في إنقاذ الأرض.

## المتطلبات

- Python 3.x أو أعلى.

## كيفية التشغيل

1. **تحميل المشروع**:
   قم بتحميل أو استنساخ هذا المستودع على جهازك باستخدام الأمر:
   ```bash
   git clone https://github.com/omarmhmd1/Adventure-Game-Python.git
   ```
2. **تشغيل اللعبة**:
بعد تحميل المشروع، يمكنك تشغيل اللعبة باستخدام Python من خلال:
   ```bash
   python adventure_game.py
   ```


3. **التفاعل مع اللعبة**:
اتبع التعليمات المعروضة على الشاشة في كل مرحلة من مراحل اللعبة لاتخاذ القرارات.

## القصة

في عالم اللعبة، يتم توجيه اللاعب لتحديد كيفية التعامل مع تهديد الفضائيين. القصة تتمحور حول الهجوم على الفضائيين باستخدام مركبة فضائية أو ترك المهمة لشخص آخر. يمكنك الهبوط على كواكب مختلفة، مواجهة وحوش، حل ألغاز لتدمير الأسلحة الليزرية الفضائية، والكثير من التحديات الأخرى.

## مميزات اللعبة

*قرارات تفاعلية*: اختياراتك ستحدد مصيرك.

*سيناريوهات متعددة*: لديك العديد من الخيارات والأحداث التي يمكن أن تؤدي إلى نهايات مختلفة.

*ألغاز*: يجب عليك حل الألغاز لتدمير المدافع الليزرية.

*نظام النقاط*: يتم منحك أو خصم النقاط بناءً على قراراتك.
